# LTO HRIS — Front-end Demo

This is a front-end only scaffold for a modern HRIS built with PHP (templating) and CSS.

To run locally with XAMPP:

1. Place the `LTO HRIS` folder in your `htdocs` (already assumed).
2. Start Apache in XAMPP.
3. Visit `http://localhost/LTO%20HRIS/` or `http://localhost/LTO HRIS/` depending on your setup.

Files added:
- `index.php`, `login.php`, `dashboard.php`, `employees.php`, `profile.php`
- `includes/` (header, footer, sidebar)
- `assets/css/style.css`, `assets/js/main.js`

This is front-end only — form submissions and data persistence are not implemented.
